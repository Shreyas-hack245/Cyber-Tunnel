# CyberTunnel

CyberTunnel is a cybersecurity-focused VPN monitoring and threat intelligence platform designed to combine secure VPN tunneling, firewall automation, real-time monitoring, and SOC-style visibility into a single modern security solution.

Built using Flask, WireGuard, Linux firewall technologies, and real-time dashboard monitoring, CyberTunnel provides a practical approach to network security analysis and threat detection.

---

## Key Features

### Secure VPN Infrastructure
- WireGuard VPN integration
- Encrypted traffic tunneling
- Secure network communication

### Threat Detection & Monitoring
- Malicious IP reputation analysis
- Threat intelligence integration
- Suspicious traffic identification
- Real-time monitoring dashboard

### Firewall Automation
- Automatic IP blocking using iptables
- Firewall rule management
- Threat logging system

### SOC Dashboard
- Professional dark-themed security dashboard
- Live monitoring interface
- Threat analytics visualization
- Real-time security log updates
- Interactive monitoring panels

### Authentication System
- Secure login authentication
- Session-based access control
- Protected dashboard routes
