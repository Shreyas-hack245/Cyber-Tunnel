# 🛡️ CyberTunnel

CyberTunnel is a privacy-focused cybersecurity platform built using WireGuard, Kali Linux, and GitHub Actions. It provides encrypted VPN tunneling, DNS leak protection, malicious traffic detection, geo-IP monitoring, and secure peer management for safe and monitored remote communication.

## Features
- Encrypted VPN tunneling using WireGuard
- Automated deployment with GitHub Actions
- DNS leak protection
- Threat and malicious IP monitoring
- Geo-IP monitoring and IP location tracking
- Secure peer management
- Network traffic analysis
- Linux firewall integration
